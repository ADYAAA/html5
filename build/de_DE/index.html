HTML5 ★ Boilerplate

ein grundsolider Standard für HTML5

HTML5 Boilerplate ist die HTML/CSS/JS-Vorlage für knallharte Profis: die Basis für eine schnelle, robuste und zukunftssichere Seite.

In mehr als Zwei Jahren iterativer Entwicklung, haben wir die bewährtesten Vorgehensweisen schon eingebaut: Cross-Browser-Normalisierung, Performance-Optimierungen und sogar optionale Features wie cross-domain AJAX und Flash. Die Vorlage für die Apache-.htaccess-Konfigurationsdatei kommt mit praktischen Caching-Regeln daher, bereitet Deine Seite für HTML5-Video vor erlaubt Dir einfache @font-face-Nutzung und gzip-Auslieferung Deines Contents.

Boilerplate ist weder ein Framework, noch schreibt es dir eine Entwicklungsphilosophie vor. Es ist eine Sammlung von Tips und Tricks, die Dir hilfen, deine Projekt schnellstmöglich und flexibel auf die Beine zu stellen.

What’s New in 0.9.5

  • Removed -webkit-font-smoothing: antialiased; it makes monospace too thin.
  • IE conditional classes have moved from the <body> tag to the <html> tag.
  • Dropped text-rendering: optimizeLegibility as it breaks small-caps and looks odd on Linux machines.
  • Added a IE6 call for the minified dd_belatedpng.
  • Revised viewport declaration to allow user scaling and clear Webkit console errors.
  • Updated Modernizr to 1.6.
  • Added web.config file for Microsoft IIS.
  • Beta release of the Build Script.
  • New project scaffolding bash script and AIR app.
  • .... and over 100 other fixes and improvements.
